#  Chatbot using Hugging Face Transformers

##  Project Overview
This project demonstrates a simple conversational AI chatbot built using a pre-trained transformer model (DialoGPT) from Hugging Face.

The chatbot interacts with users, understands queries, and generates meaningful responses in natural language.

---

##  Objectives
- Build a chatbot using transformer models
- Understand text generation using NLP
- Maintain conversational flow

---

##  Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

---

## Features
- Interactive chatbot
- Continuous conversation
- Handles user queries intelligently
- Exit command support (`exit`, `quit`)
- Clean and structured output

---

Chatbot: Hello! I am your AI assistant. How can I help you today?
